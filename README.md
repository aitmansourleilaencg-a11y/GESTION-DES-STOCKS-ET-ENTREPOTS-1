# GESTION-DES-STOCKS-ET-ENTREPOTS-1
Projet de prévision de la demande visant à optimiser la gestion des stocks et le réapprovisionnement des entrepôts. À partir de données de ventes (Kaggle M5 + données simulées WMS), des modèles comme SARIMA, Prophet et XGBoost sont utilisés pour prédire les quantités par SKU sur 30 jours et réduire les coûts de surstock et de rupture.
